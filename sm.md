# 超简版本地记账软件功能清单
完全移除独立查账模块，仅保留「记账录入 + 图表统计 + 基础配置」三大核心模块，业务逻辑极简，开发量极低，核心满足“记录收支、直观查看消费去向”的需求。

## 一、收支记账录入
单页面完成所有录入操作，无独立账单列表、无搜索筛选，仅保留最基础的记录增改能力。
1. **两种记账类型**
    - 支出录入：必填项为金额、支出分类；选填项为日期（默认当日，可修改）、文字备注
    - 收入录入：切换类型后录入，必填项为金额、收入分类；选填项为日期、备注
2. **预设分类**
    - 支出默认分类：餐饮、交通、购物、居家、娱乐、医疗、教育、其他
    - 收入默认分类：工资、兼职、红包、其他
3. **轻量记录修正**
    录入页底部仅展示最近10条记账记录，支持单条点击修改、删除，仅用于修正错记漏记，不提供查询、筛选、分页等查账能力。

## 二、消费统计图表（核心功能）
单独统计页面，基于全量本地数据计算生成，无复杂筛选逻辑，仅保留固定时间维度切换。
1. **顶部核心汇总**
    固定展示当前时间范围的总支出、总收入、结余三个数值。
2. **支出分类占比饼图**
    直观展示各支出分类的金额与占比，分类配色与设置中的颜色一一对应，直接看清主要消费去向。
3. **支出趋势折线图**
    按时间维度展示支出金额变化：查看周/月数据时按天展示，查看年数据时按月展示，固定逻辑无需自定义配置。
4. **时间切换**
    仅提供今日、本周、本月、本年4个固定选项切换，不支持自定义时间段，最大化简化开发。

## 三、基础设置与本地数据
1. **分类自定义**
    支持新增、修改、删除支出/收入分类，可为每个分类设置显示颜色（对应饼图配色），无自定义图标等复杂样式。
2. **本地备份与恢复**
    仅保留手动操作：可手动导出全量数据为本地备份文件，也可选择本地备份文件一键恢复数据，无自动备份、定时备份逻辑。
3. **基础显示设置**
    支持浅色/深色主题切换。
